Welcome To Strike Goal

Strike Goal is an web application developed for providing online tutorials in the subjects which are helpful for preparing Govt Examinations. The main aim of this application is to help an individual appearing for government job. This Application also provides the updated details and notification about the government jobs. We provide video lectures, mock tests, previous papers and materials in the form of pdf on every topic and the doubts of the users can be clarified.
